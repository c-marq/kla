## üß† **Neurodivergent-Friendly Design Principles**

### **1. Predictability**
Every action has:
- Clear button with label
- Expected outcome
- Visual confirmation
- Consistent behavior

### **2. Explicit Communication**
No assumptions about:
- Icon meanings
- Technical terms
- Expected behavior
- Current state

### **3. Reduced Cognitive Load**
- Text labels reduce interpretation
- Status messages provide guidance
- Visual confirmations remove doubt
- Clear hierarchy (primary button)

### **4. Motor Accessibility**
- Larger buttons (56px standard, 72px primary)
- More spacing between elements
- Clear touch targets
- Easy to click/tap

### **5. Anxiety Reduction**
- Friendly, encouraging language
- Immediate feedback
- No mysterious states
- Clear next steps
